What Shipped

The operating loop now has real infrastructure around it.

  • External review is live. The Cloudflare reviewer portal supports private invitations, blind tasks, onboarding, resumable sessions, and controlled export.
  • Reviewer guidance is real. A versioned onboarding video explains the evidence-first labeling rules before a reviewer touches the packet.
  • Grok 4.5 is in the comparison set. A second model labeler can now be measured against the same human reference rows.
  • Detector evidence is available. Crops and detector pointers help find difficult objects without being promoted to semantic truth.

Evaluation Safeguards

More measurement discipline, fewer unearned claims.

flowchart LR
  A[Collect] --> B[Split by time]
  B --> C[Lock held-out test]
  C --> D[Human review]
  D --> E[Compare models]
  E --> F[Audit provenance]
  F --> G[Promote or report only]
  G --> D

The loop now fails closed when artifacts, seeds, split identity, or paired evidence are incomplete. Held-out mistakes cannot silently become training data.

Human Evidence

The human review path is now usable outside the local machine.

  • Blind by design. Reviewers see the image evidence and task instructions, not prior labels, answer keys, or model scores.
  • Resumable by design. A reviewer can stop and return to the same browser session while the invitation remains active.
  • Still a qualification step. A completed packet must be imported, scored, and checked before it can support a gold-label decision.

Next Gate

Finish the independent evidence before scaling the models.

  • Complete the active reviewer packet. The latest readout is 8 of 24 saved labels.
  • Qualify a second independent reviewer. Agreement must be measured blind on the difficult container-truck cases.
  • Build benchmark v2. Use fresh operational examples and keep the held-out test set isolated.
  • Then train again. The next model run should answer a clear question against a benchmark we trust.

PortMind is not trying to win by labeling everything blindly. It is building a repeatable way to know which labels deserve belief.
